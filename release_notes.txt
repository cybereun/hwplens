### v1.0.3 (2026-06-15) - 한글 USB 이름 깨짐 현상 수정

* **한글 USB 드라이브 볼륨명 깨짐 오류 해결**:
  - 한국어 윈도우 환경에서 USB 드라이브나 디스크 볼륨명이 한글일 때 글자가 깨져서 표시되던 오류를 수정하였습니다.
  - PowerShell 출력 인코딩을 UTF-8로 지정하여 볼륨명이 정상적으로 한글로 수집되도록 수정했습니다.
* **포트 유연화 및 개선**:
  - 포트 설정을 환경 변수(`process.env.PORT`)로 지정할 수 있도록 유연하게 개선하였습니다.

### v1.0.0 (2026-06-14) - 최초 정식 릴리즈 (Initial Release)

HwpLens(한글 렌즈)는 Windows 11 탐색기 스타일의 빠르고 가벼운 한글 문서(.hwp, .hwpx) 전용 뷰어 및 관리 프로그램입니다.

**✨ Core 기능 구현**
* WASM 파서 연동 실시간 HWP / HWPX 초고속 문서 뷰어 엔진 탑재
* 페이지 줌(폭 맞춤, 쪽 맞춤, 30% ~ 200% 조절) 기능 구현

**🎨 UI/UX 디자인 완성**
* Windows 11 탐색기 순정 레이아웃 재현 및 좌측 스플리터 드래그 크기 조절 구현
* 윈도우 11 아크릴 플루언트 컨텍스트 메뉴 스타일 및 3D 공식 HwpLens 앱 아이콘 적용
* 개선된 토스트(알림창) UI 적용

**💻 파일 관리 및 뷰 기능 강화**
* 파일/폴더 CRUD 명령어 지원 및 클라이언트 검색/다차원 정렬 기능
* 드래그 앤 드롭 파일 이동/복사 바인딩
* 순정 8가지 보기 방식(아주 큰 아이콘, 타일, 내용 등) 실시간 화면 렌더러 전환 구축
* 암호 걸린 보안 문서 자동 감지 및 뷰어 한글 안내 모달 우회 기능 탑재
