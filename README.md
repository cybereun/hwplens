# HwpLens - 한글 문서 전용 탐색기 및 고속 미리보기 뷰어

HwpLens(한글 렌즈)는 Windows 11 파일 탐색기 스타일의 친숙한 레이아웃을 통해 로컬 한글 문서(.hwp, .hwpx)를 신속하게 탐색, 미리보기하고 관리할 수 있도록 설계된 한글 문서 특화 자산 관리 어플리케이션입니다.

---

## 1. 개발 계기 (Motivation)

일상적인 업무나 학업 중 수많은 한글 문서가 쌓이게 됩니다. 그러나 특정 파일에 기록된 본문 내용을 찾기 위해서는 **문서를 일일이 실행하여 열고 닫는 단순 반복 작업을 지속해야만 했습니다.** 이는 엄청난 시간 낭비일 뿐만 아니라 로컬 PC의 시스템 자원(CPU, 메모리)을 과도하게 소모시킵니다.

HwpLens는 **"수많은 한글 문서의 내용을 가장 효율적으로 빠르게 훑어볼 수 있는 방법은 없을까?"**라는 의문에서 시작되었습니다.
우리에게 친숙한 Windows 탐색기와 똑 닮은 구조를 바탕으로, 한글 프로그램이나 별도의 오피스 무거운 뷰어 프로그램을 띄우지 않고도 한글 문서(.hwp, .hwpx)만을 자동으로 분류하여 단일 클릭만으로 본문 내용을 브라우저/앱 화면에 즉시 렌더링해 줍니다. 

여기에 파일 이동, 복사, 삭제 등 탐색기 본연의 관리 기능과 로컬 한글 프로그램과의 자연스러운 연동을 결합하여 파일 탐색 효율을 극대화했습니다.

## 2. 주요 특징 (Key Features)

* **초고속 웹 미리보기**: 독자적인 WebAssembly(WASM) 파서 코어(`rhwp`)를 내장하여, 한글 프로그램을 무겁게 띄우지 않고도 HWP 5.0 및 HWPX 문서의 텍스트와 레이아웃을 SVG 기반으로 브라우저에 실시간 렌더링(세로 스크롤 미리보기)합니다.
* **직관적인 윈도우 11 스타일 파일 관리 (CRUD)**:
  * 마우스 우클릭을 통한 윈도우 11 순정 스타일 아크릴 컨텍스트 메뉴 지원.
  * 단축키 연동 및 실제 파일 복사, 잘라내기, 붙여넣기, 이름 바꾸기(F2/인라인), 삭제(Delete 키) 물리 파일 핸들링.
  * 파일 및 폴더 드래그 앤 드롭(Drag & Drop) 이동 및 복사 연동.
* **다양한 보기 모드 지원 (실제 작동)**: 윈도우 탐색기 순정 보기 방식 8가지(아주 큰 아이콘, 큰 아이콘, 보통 아이콘, 작은 아이콘, 목록, 자세히, 타일, 내용)의 레이아웃 실시간 리플로우 및 파일 정보 바인딩 완벽 작동.
* **사용자 세션 연동 실행**: 미리보기로 훑어본 문서를 더블클릭하거나 컨텍스트 메뉴의 '열기(Enter)'를 누르면, 사용자 윈도우에 연결된 기본 한글 프로그램(한컴오피스)을 즉시 팝업으로 호출하여 편집 모드로 이어갈 수 있습니다.
* **다차원 파일 정렬 및 필터**: 파일 이름, 수정한 날짜, 유형, 파일 크기를 기준으로 실시간 오름차순/내림차순 정렬 및 상단 검색창을 통한 키워드 실시간 매칭 필터링.
* **강력한 보안 문서 예외 처리**: 문서 열람 중 암호(Password)나 암호화가 감지될 경우, 브라우저가 정지하거나 오류를 뱉는 대신 *"비밀번호가 설정된 암호화 문서입니다. 웹 미리보기를 지원하지 않으므로, 문서를 더블클릭하여 데스크톱 한글 프로그램에서 직접 확인해 주십시오."*라는 친절한 한글 안내 화면을 띄워 안전하게 로컬 실행으로 우회하도록 돕습니다.

## 3. 프로그램 설치 방법 (Installation)

HwpLens는 개발자용 로컬 웹 서버 환경 구동과 엔드유저용 실행파일(.exe) 설치 패키징 두 가지 방식을 모두 지원합니다.

### 3.1 로컬 개발자 환경 구동
1. **Node.js 설치**: 시스템에 Node.js(v18 이상 권장)가 설치되어 있는지 확인합니다.
2. **저장소 클론 및 진입**:
   ```bash
   cd C:\Users\j.u.Eun\hwpview_temp
   ```
3. **종속성 패키지 설치**:
   ```bash
   npm install
   ```
4. **WASM 핵심 자산 셋업**:
   ```bash
   npm run setup
   ```
   * 이 명령은 `node_modules` 내부의 핵심 WebAssembly 모듈을 프론트엔드가 서비스하는 `public/` 디렉토리로 동기화합니다.
5. **서버 시작**:
   ```bash
   npm start
   ```
   * 서버 구동 완료 후 브라우저를 열고 `http://localhost:8800`에 접속합니다.

### 3.2 윈도우 단독 실행파일(.exe) 설치 패키징
1. 일렉트론 빌드 전용 폴더인 `HwpLens_Build`로 이동합니다.
2. 패키징 빌드 명령어를 실행합니다:
   ```bash
   npm run dist
   ```
3. 빌드가 완료되면 `HwpLens_Build/dist/` 경로에 **`HwpLens Setup 1.0.0.exe`** 설치 파일이 생성됩니다.
4. 이 설치 파일을 더블클릭하여 PC에 영구 설치한 후, 바탕화면 바로가기 아이콘을 통해 단독 데스크톱 앱으로 편리하게 구동할 수 있습니다.

## 4. 상세 사용 방법 (Usage Guide)

### 4.1 기본 화면 구성
* **상단 툴바**: 탐색 창 켜기/끄기, 새로 만들기, 잘라내기/복사/붙여넣기, 이름 바꾸기, 삭제, 정렬, 보기 모드 옵션이 일목요연하게 모여 있습니다.
* **좌측 탐색 창**:
  * **즐겨찾기 아코디언**: 홈, 바탕 화면, 다운로드, 문서, 사진, 음악, 동영상 및 자동 감지된 OneDrive 폴더가 모여 있습니다. 오른쪽 화살표 아이콘을 눌러 아코디언 형태로 자유롭게 접고 펼칠 수 있습니다.
  * **내 PC**: 실제 시스템의 드라이브 계층 구조를 트리 노드로 렌더링하며 클릭하여 하위 폴더를 실시간 전개합니다.
* **중앙 파일 리스트**: 선택된 폴더 내의 한글 문서(.hwp, .hwpx)와 하위 폴더들을 실시간 정렬하여 노출합니다.
* **우측 미리보기 뷰어**: 선택된 한글 파일의 상세 메타데이터(크기, 페이지 수, 수정 날짜 등)와 함께 본문 페이지를 고해상도 SVG로 나열합니다.

### 4.2 파일 탐색 및 미리보기
* 가운데 파일 목록에서 한글 파일을 **마우스 단일 클릭**하면, 우측 뷰어에 첫 페이지부터 차례대로 본문이 나타납니다.
* 뷰어 상단의 **페이지 컨트롤**(`◀` `1 / 5` `▶`) 및 마우스 휠 스크롤을 사용하여 페이지를 탐색할 수 있습니다.
* **배율 맞춤 기능**을 사용해 `폭 맞춤`(화면 너비에 종이를 맞춤) 또는 `쪽 맞춤`(화면 높이에 맞춰 한 페이지가 한눈에 들어오게 함) 버튼을 누르면 실시간으로 배율이 조절되며, 축소/확대 버튼으로 `30%`부터 `200%`까지 미세 배율 조정도 가능합니다.

### 4.3 파일 및 폴더 관리 (Windows 11 순정 스타일)
* **컨텍스트 메뉴**: 파일 또는 폴더를 우클릭하면 나타나는 아크릴 스타일 메뉴를 통해 다음 명령을 내릴 수 있습니다:
  * **열기 (Enter)**: 폴더는 진입하고, 파일은 연결 프로그램(한글)으로 바로 엽니다.
  * **잘라내기 (Ctrl+X) / 복사 (Ctrl+C)**: 대상을 클립보드에 담습니다.
  * **붙여넣기 (Ctrl+V)**: 빈 영역이나 다른 폴더 위에 붙여넣어 실제 파일 시스템 상에 이동/복사합니다.
  * **이름 바꾸기 (F2)**: 상세 보기 모드에서는 파일명 텍스트 상자에서 바로 이름을 수정하는 인라인 변경을, 기타 보기 모드에서는 정밀 팝업 창을 제공합니다.
  * **삭제 (Delete)**: 해당 파일을 로컬 하드디스크에서 영구히 삭제합니다.
  * **즐겨찾기에 추가 / 경로 복사 (Ctrl+Shift+C) / 속성 보기 (Alt+Enter)**: 파일의 절대 경로를 클립보드에 복사하거나 윈도우 스타일의 속성 모달로 상세 메타데이터를 볼 수 있습니다.
* **드래그 앤 드롭**: 중앙 영역의 파일/폴더를 마우스 드래그하여 좌측의 즐겨찾기 폴더나 하위 폴더 노드 위에 놓으면 파일이 실시간 이동하며, `Ctrl` 키를 누른 채 드롭하면 복사됩니다.

### 4.4 정렬 및 보기 방식 제어
* **정렬**: 상단 [정렬] 단추를 누르면 드롭다운이 나타나며 **이름, 수정한 날짜, 유형, 크기** 기준 정렬 및 **오름차순/내림차순** 전환이 가능합니다.
* **보기 방식 전환**: 상단 [보기] 단추를 클릭하여 원하는 크기의 아이콘 뷰나 리스트, 내용 뷰로 전환할 수 있습니다. 각 뷰 방식은 실제 사용자의 화면 리스트 형태와 카드 레이아웃을 실시간 변경시킵니다.

## 5. 업데이트 기록 (Release Notes)

### v1.0.2 (2026-06-14) - 핫픽스 (포트 충돌 오류 해결)
* **프로그램 중복 실행 방지 및 에러 해결**:
  * 프로그램 중복 실행 시 `EADDRINUSE` 자바스크립트 오류 팝업이 뜨는 문제 해결.
  * 새 창을 여는 대신 기존에 띄워둔 HwpLens 창을 최상단으로 안전하게 다시 불러오도록 Single Instance Lock 로직 개선.

### v1.0.1 (2026-06-14) - 미리보기 썸네일 기능 추가
* **탐색기 썸네일 뷰어 기능 탑재**:
  * 아주 큰 아이콘 / 큰 아이콘 보기 모드 시 한글 문서(.hwp, .hwpx)의 1페이지 미리보기 썸네일을 아이콘으로 즉시 렌더링.
  * IntersectionObserver 지연 로딩(Lazy Loading)을 적용하여 시야에 보이는 파일만 메모리에 올려 퍼포먼스 최적화.
  * 동시성 제어 큐(Queue) 및 메모리 캐싱을 통해 브라우저 스레드 정체 현상 완벽 방어.
  * 썸네일 변환 과정에서 유려한 빛 반사 로딩 애니메이션(Shimmer) 효과 추가.

### v1.0.0 (2026-06-14) - 최초 정식 릴리즈 (Initial Release)
* **Core 기능 구현**:
  * WASM 파서 연동 실시간 HWP / HWPX 초고속 문서 뷰어 엔진 탑재.
  * 페이지 줌(폭 맞춤, 쪽 맞춤, 30% ~ 200% 조절) 기능 구현.
* **UI/UX 디자인 완성**:
  * Windows 11 탐색기 순정 레이아웃 재현 및 좌측 스플리터 드래그 크기 조절 구현.
  * 윈도우 11 아크릴 플루언트 컨텍스트 메뉴 스타일 및 3D 공식 HwpLens 앱 아이콘 적용.
* **파일 관리 및 뷰 기능 강화**:
  * 파일/폴더 CRUD 명령어 지원 및 클라이언트 검색/다차원 정렬 기능.
  * 드래그 앤 드롭 파일 이동/복사 바인딩.
  * 순정 8가지 보기 방식(아주 큰 아이콘, 타일, 내용 등) 실시간 화면 렌더러 전환 구축.
  * 암호 걸린 보안 문서 자동 감지 및 뷰어 한글 안내 모달 우회 기능 탑재.

## 7. 자주 묻는 질문 및 문제 해결 (FAQ & Troubleshooting)

### Q. 설치 후 실행했더니 `A JavaScript error occurred in the main process (listen EADDRINUSE :::8800)` 에러가 뜹니다.
이 에러는 HwpLens가 내부 통신용으로 사용하는 **8800번 포트**를 이미 다른 프로세스가 점유하고 있을 때 발생합니다. 

**해결 방법:**
1. **버전 업데이트(v1.0.2 이상 권장)**: `v1.0.2` 버전부터는 이미 실행 중일 때 에러 창을 띄우지 않고 기존 창을 불러오도록 개선되었습니다. 가급적 최신 버전 다운로드를 권장합니다.
2. **이전 프로세스 강제 종료**: 기존 버전(v1.0.1 등)을 사용하다 지우고 다시 설치하는 과정에서 백그라운드에 기존 프로세스가 완전히 종료되지 않고 남아있을 수 있습니다. 
   - 키보드에서 `Ctrl + Shift + Esc`를 눌러 **작업 관리자**를 엽니다.
   - 목록에서 `HwpLens` 또는 `Node.js` 관련된 프로세스가 켜져 있다면 우클릭하여 **작업 끝내기(종료)**를 누르신 후, 앱을 다시 실행해 보세요.
3. **포트 충돌 확인**: 매우 드물게 다른 프로그램(개발용 웹 서버 등)이 8800번 포트를 선점하고 있을 수 있습니다. 해당 프로그램을 잠시 종료하시기 바랍니다.
